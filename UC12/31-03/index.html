Sobre html

O que é html

HTML (HyperText Markup Language ou Linguagem de Marcação de Hipertexto) é a linguagem fundamental usada para estruturar e dar sentido ao conteúdo de páginas web, funcionando como o "esqueleto" de um site.
Ela organiza elementos como textos, títulos, imagens e links, sendo interpretada pelos navegadores.
